## 🔧 Problemas encontrados

| Error | Causa | Solución |
|---|---|---|
| `[Errno 98]` puerto ocupado | LiteLLM previo en background | `pkill -f litellm` |
| `git pull` abortado | Cambios locales en yaml | `git checkout -- litellm-config.yaml` |
| Groq 401 | API key caducada | Cambiar principal a Gemini |
| Gemini 429 | Free tier agotado por llamadas de agente | Cambiar principal a DeepSeek V3 |
| Cerebras/Groq `ContextWindowExceeded` | Límite 8K tokens | Eliminar de fallbacks del principal |
| Prompt escrito en bash | Confusión terminal vs OpenCode | Abrir `opencode` primero |
