## 🎯 Para qué sirve este repo exactamente

### 1. Sistema de agentes que trabajan para Álvaro
Agentes IA (Aider, Claude Code, CrewAI) que gestionan, mejoran y escalan todos los repos de forma automática. No herramientas que hay que usar a mano — agentes que trabajan solos con órdenes de alto nivel.

### 2. Motor del portfolio
Cada experimento, cada investigación, cada agente construido aquí **es portfolio real**. No proyectos de ejemplo — casos reales de IA aplicada que se pueden mostrar en entrevistas.

### 3. Laboratorio de IA aplicada
Investigación verificada de modelos, benchmarks, estrategias. Lo que funciona va al registro. Lo que no funciona también. Toda la experimentación con IA es un activo diferenciador para buscar trabajo.

### 4. Puente entre THDORA y el mundo
Lo que se aprende en THDORA (vida real) se abstrae aquí como conocimiento generalizable. Lo que se experimenta aquí se aplica primero en THDORA.

## 🤖 El sistema de agentes

El núcleo de este repo: agentes que trabajan para Álvaro.

| Agente | Herramienta | Función |
|---|---|---|
| **Agente de código** | Aider + Groq | Fix bugs, mejoras en cualquier repo |
| **Agente de arquitectura** | Claude Code + Devstral 2 | Refactorizaciones grandes, diseño |
| **Agente de investigación** | DeepSeek R1 + OpenRouter | Investigar tecnologías, comparar opciones |
| **Agente de automatización** | n8n + Groq | Workflows, briefs, recordatorios |
| **Agente de portfolio** | Claude Code + Qwen3 | Actualizar docs, READMEs, demos |

Ver configuración detallada en [`agentes/`](agentes/)

## 🏆 Por qué esto es un diferenciador para buscar trabajo

En 2026, la diferencia entre un desarrollador normal y uno que destaca es exactamente esto:

- **No usas IA** → estás por debajo del mercado
- **Usas IA como herramienta** → estás en el mercado
- **Has construido un ecosistema de agentes que trabajan para ti** → estás por encima del mercado

Este repo es la prueba de que Álvaro está en la tercera categoría. No se trata de decir “sé usar ChatGPT”. Se trata de mostrar un sistema real, documentado, en producción, con experimentos verificados y resultados medibles.
