# 🤖 AI Toolkit — Ecosistema de Agentes IA (Coste = 0€)

> Stack completo open source para programar, automatizar y vivir mejor con IA.
> Sin pagar suscripciones. Sin correr modelos en local. Todo en la nube gratuita.

**Por [alvarofernandezmota-tech](https://github.com/alvarofernandezmota-tech) — documentado en español para la comunidad hispana.**

---

## ⭐ Por qué este repo

Cada herramienta de IA de pago tiene una alternativa gratuita igual de potente. Este repo documenta cómo montar un ecosistema completo de agentes que trabajan para ti: en tu código, en tu vida, en tu información.

**Coste real mensual: 0€.** Consumo de tu ordenador: <1% CPU, ~360 MB RAM.

---

## 🚀 Empezar en 1 comando

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/alvarofernandezmota-tech/ai-toolkit/main/setup.sh)
```

Instala: Python, Aider, Node.js, Claude Code, THDORA, Docker y n8n. Menú interactivo para elegir qué quieres.
