# 💬 Prompts para Aider y Claude Code

Prompts que funcionan bien para sacar el máximo a los modelos gratuitos.

## Para empezar un proyecto nuevo

```
Voy a crear [descripción del proyecto]. 
Tecnologías: [lenguaje/framework].
Empecemos por la estructura de carpetas y el archivo principal.
Hazlo modular y con buenas prácticas desde el principio.
```

## Para debugging

```
Este código da el siguiente error: [pegar error]
Explica por qué ocurre y corrigiéndolo con el menor cambio posible.
```

## Para aprender mientras construyes

```
Antes de escribir el código, explícame en 3 líneas qué patrón vas a usar y por qué.
Luego escribe el código.
```

## Para code review

```
Revisa este código y dime:
1. Un problema de rendimiento si existe
2. Un problema de seguridad si existe
3. Una mejora de legibilidad
Solo esos tres puntos, sin más.
```

## Para documentar

```
Escribe el README de este proyecto en español.
Incluye: qué hace, instalación en 3 pasos, ejemplo de uso.
Tono técnico pero accesible para alguien que está aprendiendo.
```
