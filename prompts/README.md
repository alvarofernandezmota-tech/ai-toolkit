# Prompts

> Prompts reutilizables probados en producción.

## Estructura

```
prompts/
├── codigo/       ← prompts para tareas de código
├── research/     ← prompts para investigación
├── agentes/      ← system prompts para agentes
└── utilidades/   ← prompts de uso general
```

## Reglas

- Cada prompt incluye: propósito, modelo recomendado, ejemplo de uso y resultado esperado
- Si un prompt falla con ciertos modelos, documentarlo
- Los mejores prompts se mueven a `opensource/` para compartir con la comunidad
