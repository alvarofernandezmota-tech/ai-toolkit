## Prompts básicos para Claude Code

```
# Diagnóstico inicial de un repo
> Lee toda la estructura del proyecto y dime qué hay, qué falta y qué mejorarías primero

# Documentar arquitectura
> Genera ARCHITECTURE.md documentando el patrón actual del proyecto

# Revisar calidad
> Revisa todo el código y dime los 3 problemas más importantes a resolver

# Fix autónomo
> Arregla el bug en [fichero]: [descripción]. Haz el commit cuando termines.
```
