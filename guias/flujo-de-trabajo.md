# 📝 Flujo de trabajo diario

Cómo uso este stack para programar, aprender y escalar proyectos.

## Sesión de coding típica

```
1. Abro el terminal
2. Entro al proyecto: cd mi-proyecto
3. Arranco Aider con Ollama (sin coste):
   aider --model ollama/gemma3:4b
4. Le describo lo que quiero hacer
5. Aider edita los archivos, yo reviso
6. Git commit automático por cada cambio
7. Push a GitHub → portfolio actualizado
```

## Cuándo usar cada herramienta

| Situación | Herramienta |
|---|---|
| Coding del día a día | Aider + Ollama local |
| Proyecto complejo con agente | Claude Code + OpenRouter |
| Tarea puntual grande | Arena AI (gratis en browser) |
| Automatización de vida | n8n local |
| Sin internet | Ollama puro en terminal |
