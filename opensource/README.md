# Opensource — Contenido para la Comunidad

> Esta carpeta contiene herramientas, scripts y documentación preparados para compartir con la comunidad open source.

## Estado actual

🚧 En construcción — estamos madurando el proyecto internamente antes de publicar.

## Qué vendrá aquí

- Scripts de arranque limpios y comentados para uso general
- Guías de setup reproducibles (Ollama + LiteLLM + OpenCode)
- Configuraciones de ejemplo sin datos personales
- Documentación de patrones que funcionan

## Criterio para mover algo aquí

1. ✅ Probado y documentado en `pruebas/`
2. ✅ Sin rutas absolutas ni API keys
3. ✅ Comentado para que cualquiera lo entienda
4. ✅ README propio que explica cómo usarlo

## Contribuir

Este proyecto es personal por ahora. Cuando esté listo para la comunidad, se publicará con licencia MIT.
