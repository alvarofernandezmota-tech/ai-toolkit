# Investigación

> Research verificado con fuentes sobre IA, modelos, herramientas y estrategias.

## Qué hay aquí

Documentación de investigación propia: comparativas de modelos, análisis de herramientas, experimentos y conclusiones.

## Estructura

```
investigacion/
├── modelos/      ← comparativas y benchmarks
├── herramientas/ ← análisis de tools (OpenCode, Aider, n8n...)
└── estrategia/   ← decisiones de arquitectura y por qué
```

## Formato

Cada documento incluye: fecha, fuentes, conclusión práctica y si aplica a nuestro stack.
