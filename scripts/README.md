# ⚙️ Scripts

Scripts de automatización del ecosistema.

## Scripts disponibles

| Script | Para qué |
|---|---|
| `../setup.sh` | Bootstrap completo del ecosistema |

## Cómo añadir un script

1. Crea el fichero en esta carpeta
2. Añade permisos: `chmod +x tu-script.sh`
3. Documenta aquí para qué sirve
4. Si es reutilizable, añade ejemplos de uso

## Convenciones

- Nombre en `kebab-case.sh`
- Primera línea siempre `#!/bin/bash`
- `set -e` para parar en errores
- Comentarios en español
